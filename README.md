# HW-Info
<p align="center">CLI-скрипт для сбора информации о характеристиках системы.</p>

HW-Info это командная утилита для сбора информации о системе, написанная на Python3.7+.

HW-Info показывает информацию о вашей операционной системе, программном и аппаратном обеспечении.

HW-Info базируется на bash скрипте <a href='https://github.com/dylanaraps/neofetch'>neofetch</a>, поэтому для обеспечения работоспособности HW-Info необходимо иметь заранее установленный <a href='https://github.com/dylanaraps/neofetch'>neofetch</a>.

HW-Info дополняет информацию, выводимую neofetch, а также позволяет использовать ее в своих Python-скриптах, имея возможность использовать себя как Python-модуль.
